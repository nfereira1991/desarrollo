¿Que es la gestión y administración completa del Help Desk EndPoint? 

Endpoint Security o Endpoint Protection es un enfoque centralizado para administrar y proteger todos los puntos finales (servidores, computadoras de escritorio, computadoras portátiles, teléfonos inteligentes etc.) conectados a la red de TI corporativa.

Esta metodología permite una gestión de seguridad eficiente, eficaz y más sencilla.  

ADVANCED ENDPOINT PROTECTION

Proporcione protección avanzada de puntos finales a través de una plataforma de seguridad de puntos finales integrada
 
Comodo AEP protege todos sus servidores, computadoras de escritorio, computadoras portátiles y dispositivos móviles de malware conocido y desconocido, sin necesidad de firmas o actualizaciones.
 
Puede ver y administrar todo a través del panel único de nuestra plataforma de administración unificada.

Comodo AV

Un motor antivirus proactivo que detecta y elimina automáticamente virus, gusanos y otro malware.                         

Contención 

La contención de amenazas de Comodo proporciona una protección total contra las amenazas de día cero sin afectar la experiencia del usuario final o los flujos de trabajo. 

Valkyrie

Valkyrie es un servicio de veredicto de archivos basado en la nube que prueba archivos desconocidos con una variedad de comprobaciones estáticas y de comportamiento para identificar aquellos que son maliciosos.

Firewall

Firewall de filtrado de paquetes altamente configurable que defiende constantemente los ataques de Internet entrantes y salientes.

Web URL Filtering

Una potente interfaz de configuración de roles que puede crear reglas tan granulares como sea necesario, incluso por usuario.

Virusscope

Monitorea las actividades de los procesos que se ejecutan en los puntos finales y genera alertas si toman acciones que podrían amenazar la privacidad y seguridad del usuario final.

Administración de dispositivos móviles

Comodo IT and Security Manager permite a los administradores de red configurar de forma remota los derechos de acceso al dispositivo, la configuración de seguridad, las preferencias generales y monitorear y administrar el dispositivo.

Puntos finales de Windows

Comodo IT and Security Manager permite a los administradores instalar Comodo Endpoint Security, administrar la configuración de seguridad en ellos, ver y administrar aplicaciones instaladas, ejecutar análisis antivirus, administrar la actualización del sistema operativo  y más.

Comodo Client Security (CCS) 

Es el software de seguridad de punto final administrado de forma remota instalado en dispositivos Windows administrados. . Cada componente de CCS se puede configurar para ofrecer el nivel de seguridad deseado mediante la aplicación de perfiles de configuración.

HIPS

El sistema de prevención de intrusiones de host; monitorea constantemente la actividad del sistema y solo permite que se ejecuten ejecutables y procesos si cumplen con las reglas de seguridad aplicadas por el perfil de Windows aplicado a la computadora administrada.

Monitoreo y administración remota

El módulo de monitoreo y administración remotos (RMM) es una aplicación eficiente de monitoreo de puntos finales que permite a los administradores monitorear y administrar múltiples puntos finales desde una consola centralizada. 

Administración de parches

El módulo de administración de parches monitorea las actualizaciones y parches del sistema operativo Windows para el software de terceros de Windows. Identifica los parches apropiados para la versión del sistema operativo de cada dispositivo Windows administrado y si es necesario, los instala de forma automática.

La consola de administración

La Consola administrativa es el centro neurálgico de Comodo IT and Security Manager (ITSM), permitiendo a los administradores agregar o importar usuarios, inscribir dispositivos, crear grupos de dispositivos, aplicar perfiles de configuración, ejecutar escaneos de antivirus (AV) y más.

Panel de control

Contiene cuadros y gráficos que muestran la estructura y el estado de seguridad de los dispositivos en su red.

Dispositivos

Permite que los administradores administren y controlen los dispositivos inscritos, instalen aplicaciones de forma remota, generen sirenas, borren, bloqueen y apaguen los dispositivos inscritos, instalen y administren aplicaciones en dispositivos de forma remota, administren grupos de dispositivos y más.

Usuarios

Permite a los administradores crear y administrar usuarios y grupos de usuarios, inscribir sus dispositivos y asignar perfiles de configuración a los dispositivos.

Plantillas de configuración

Cree y administre los perfiles de configuración que se aplicarán a los teléfonos inteligentes y tabletas iOS y Android registrados, los puntos finales de Windows y Mac OS X.

Almacén de aplicaciones

Permite a los administradores agregar aplicaciones para enviarlas a dispositivos iOS y Android administrados.

Aplicaciones

Permite a los administradores ver y administrar aplicaciones instaladas en dispositivos Android e iOS registrados, ver archivos instalados en dispositivos Windows administrados, programas de espacio aislado, ver y administrar la lista de proveedores de software y administrar la instalación de parches del sistema operativo en dispositivos Windows administrados.

Subsistemas de seguridad

Permite a los administradores ejecutar análisis AV y actualizaciones de bases de datos de firmas de virus en los dispositivos inscritos, administrar malware identificado, ver amenazas, administrar elementos en cuarentena, ver y administrar aplicaciones contenidas y más.

Configuración

Permite a los administradores crear roles de administrador y usuario con diferentes privilegios, configurar el comportamiento de varios componentes y agentes de ITSM, renovar / actualizar licencias y más.
